# Crafting Interpreter
This implementations are from the book "Crafting Interpreters".

> [Book Link](https://craftinginterpreters.com/contents.html)

## How to build

```sh
cd src
javac tool/GenerateAst.java

java tool.GenerateAst ../src/wialang

cd ..
make
```

## Usage

```sh
./wia script.wia
```

## REPL

```sh
./wia
```
